<h3>课程介绍：</h3>
<p><a title="查看与 Ant 相关的文章" target="_blank">Ant</a> 定义生成文件之间的依赖关系,它使用跨平台的 Java 类。使用 <a title="查看与 Ant 相关的文章" target="_blank">Ant</a>，您能够编写单个生成文件，这个生成文件在任何 Java 平台上都一致地操作(因为 Ant 本身也是使用 Java 语言来实现的),这就是 Ant 最大的优势。</p>
